# HızlıMarket 🛒

**Basit barkodlu satış otomasyonu.**

## 🚀 Özellikler
- Barkod ile ürün ekleme
- Satış listesi (DataGridView)
- Ürün silme (sağ tık menü)
- Toplam tutar hesaplama
- SQLite veritabanı desteği
- Temiz kullanıcı arayüzü

## 🛠️ Kullanılan Teknolojiler
- C#
- WinForms
- SQLite
- .NET Framework
